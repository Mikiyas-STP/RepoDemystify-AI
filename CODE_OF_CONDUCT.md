# Code of Conduct

## Our Commitment

RepoDemystify AI is committed to providing a welcoming, respectful, and professional environment for everyone.

We value collaboration, learning, and constructive discussion.

---

# Expected Behaviour

Contributors should:

- Be respectful and professional
- Welcome different perspectives
- Provide constructive feedback
- Focus on improving the project
- Help create a positive community

---

# Unacceptable Behaviour

Examples include:

- Harassment
- Personal attacks
- Discriminatory language
- Trolling
- Publishing private information
- Disruptive behaviour

---

# Technical Discussions

Disagreements about:

- architecture
- implementation choices
- technology decisions

are expected.

They should be handled through:

- evidence
- respectful discussion
- engineering reasoning

---

# Enforcement

Project maintainers may take appropriate action when this code of conduct is violated.

Actions may include:

- warning
- removing inappropriate content
- temporary restrictions
- banning participation

---

# Reporting Issues

If you experience unacceptable behaviour, contact the project maintainers through the repository communication channels.

---

Thank you for helping build a respectful open-source community.
